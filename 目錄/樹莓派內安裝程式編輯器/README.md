## 安裝pycharm comunity editor for respberry4 2GB 以上

## 更新java develop kit
```
$ sudo apt update
$ sudo apt install default-jdk
```

### 下載pycharm
```
https://www.jetbrains.com/pycharm/download/
```


### 解壓縮,並copy 至 /opt 目錄
```
sudo mv pycharm-community-2020.x.x /opt/pycharm-community-2020.x.x
```

### 執行pycharm.sh

```
./bin/pycharm.sh
```
